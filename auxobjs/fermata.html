6.1. Fermatas

Fermatas are defined attached to notes and rests, using the fermata element:

fermata   ::=  (fermata placement [componentOptions*])
placement ::=  { above | below }

Examples:

(score (vers 2.0) (instrument (musicData
    (clef G)(key C)
    (n g4  q (fermata above))
    (n g4  q (fermata below))
    (r q (fermata above))
    (r q (fermata below))
    (barline end)
)))
